## The headline: the north star is more buildable — and more analog-native — than it looks

Every organ the horizon needs has a **concrete, gradient-free, analog-native first build**, and in three of four cases
the analog substrate does the hard part *for free*:

| Organ | First build (gradient-free) | What the substrate gives for free |
| ----- | --------------------------- | --------------------------------- |
| **Hippocampus** (t3.1) | **delta-rule fast-weight memory** `W += β(v−Wk)⊗k`, read `Wk`; **β = the existing P8 DDM surprise gate** | content-addressable recall = a crossbar read / analog-CAM settle (Li 2020) |
| **Recurrence** (t3.2) | **contractive weight-tied block** over frozen SCFF, iterate to fixed point; **halt = ‖Δz‖ comparator** | the fixed-point *settle* = a physical relaxation; monDEQ makes it a contractive **certificate** |
| **Self-signal** (t3.3) | **re-settle agreement** = semantic entropy run *physically* (settle k times, agreement of namer reads) | **thermal noise = the free sampler**; the namer = the meaning-equivalence class |
| **Controller** (t3.4) | **NOT YET** — premature; build only when a second organ creates a coordination failure | WTA = free slot competition; DDM gate = the write-gate |

## The build ORDER (the horizon has a critical path)

1. **Hippocampus first** (delta-rule memory) — it is the only one that adds a *new* capability, and it is the trigger for everything downstream.
2. **Recurrence** (contractive settle + free halt) — needs something to iterate *over*; the memory gives it state.
3. **Self-signal** (re-settle agreement) — needs the settle loop to exist (the re-runs are settles).
4. **Controller LAST** — and only once the hippocampus is a *second learning organ* and the settle loop must reconcile it with the namer within a step budget. Adding it earlier violates the build discipline.

## The one research question the whole north star reduces to

**A gradient-free / closed-form learning-and-routing rule.** Memory (delta-rule), recall (crossbar), settle (physics),
self-signal (noise) all have substrate-native gradient-free forms. The *coordinator's routing* does not — every published
workspace/MoE win credits **learned** routing. If the project solves streaming, gradient-free routing-that-specializes, the
last non-native piece of the horizon becomes native. **This is the flagship open problem.**
