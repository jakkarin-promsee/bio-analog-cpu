# Tier 1 — Nearest neighbors

**The closest whole-system prior art to draft-6.0** (forward-only + local + continual + on-chip). The question every
topic here answers: *who else built something like this, and exactly how are we different?* This is the highest-priority
tier — a research-first project's top need is establishing the prior art it would have to cite.

One subfolder per topic (`<topic-slug>/`), each with `_SYNTHESIS.md` + per-paper cards. See [`../BACKLOG.md`](../BACKLOG.md) for the queue.
