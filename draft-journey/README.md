# `draft-journey/` — the journey log (drafts 1.0 → 5.1 → …)

> **What this is.** The chronological **idea log**: every draft of the design as a single-file
> snapshot, from the first sketch (`draft1.0.md`) to the last pre-pivot spec (`draft5.1-full.md`) —
> including the questions (`.q`), explorations (`.x`), and revisions (`.r`) raised along the way. It is
> the *story of how the thinking walked there* — the early mental experiments, not a finished design.

This is **not** a `draftN/` design *world*. The full worlds are the versioned folders:
[`../draft5.0/`](../draft5.0/README.md) (the attribution era, now superseded) and
[`../draft6.0/`](../draft6.0/README.md) (the live SCFF + GD line). `draft/` is the trail that led to
them — read it as narrative, not as the current plan.
